# Usage

## Register Account
1. Open frontend
2. Create account
3. Verify email
4. Login

## Gameplay Flow
- Player spawns with a base
- Resources generate passively
- Capture tiles to expand territory
- Train armies using resources
- Engage other players in real-time battles

## Real-Time Events
- Tile capture updates
- Army movement
- Battle results
- Chat & notifications

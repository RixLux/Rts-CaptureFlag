# Getting Started

This guide helps you understand the project structure and prerequisites.

## Prerequisites

Make sure you have:

- Node.js (v18+ recommended)
- MongoDB (local or Atlas)
- Git
- npm

## Project Structure
```
war-territory-game/
├── client/ # Frontend (React + Vite)
├── server/ # Backend (Node + Express)
├── docs/ # Documentation
└── mkdocs.yml
```

## Tech Stack

| Layer | Technology |
|------|-----------|
Backend | Node.js, Express |
Database | MongoDB |
Real-time | Socket.IO |
Frontend | React + Vite |
Auth | JWT + Email verification |
